## 🐞 問題紀錄｜ISSUE-005
（內部編號：18653）

### 🔍 問題摘要
使用處理文件回傳圖片 API 時，傳入文字 PDF，回傳的圖片文字消失了

---

### 📖 問題說明
使用處理文件回傳圖片 API 時，當傳入內容為「含文字的 PDF 檔案」，API 回傳的圖片結果中，原本應存在的文字內容消失，僅剩背景或圖像，導致後續判讀與驗證困難。

---

### 🧪 重現步驟
1. 以上傳附檔 PDF 呼叫「處理文件回傳圖片 API」
2. 檢視 API 回傳之圖片結果
3. 可觀察到圖片中的文字未被保留
